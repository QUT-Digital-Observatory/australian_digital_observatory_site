Platform Factsheet: Twitter

Last updated: 2022-11-28

This platform-specific fact sheet is intended to provide researchers with tips and resources to get started using Twitter in their research.

Researchers should be aware that platforms can and do change over time, so for the most up to date information please refer to Twitter's official documentation. We have provided some links to Twitter's documentation below and will endeavour to keep them up to date.

Basic information about the API

Twitter's API has three access levels: Essential, Elevated, and Academic Research.

Rate limits:

  • Essential allows you to retrieve 500,000 tweets per month (only tweets from the past 7 days).
  • Elevated allows you to retrieve 2 million tweets per month.
  • Academic Research allows you to retrieve 10 million tweets per month.

See the following link for more information about Twitter API access levels and versions.

Tools

  • twarc is a command line tool and Python library for collecting an archiving Twitter JSON data via the Twitter API. It has an ecosystem of plugins for doing things with the collected data.

  • tidy_tweet tidies Twitter JSON collected with twarc into relational tables. The resulting SQLite database is ideal for importing into analytical tools, or for using as a datasource for a programmatic analytical workflow that is more efficient than working directly from the raw JSON.

  • tweetexploR is an R package for exploring and visualising tidy Twitter data created with the tidy_tweet library.
