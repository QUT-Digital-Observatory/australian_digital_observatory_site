YouTube videos of Australia

Datasets

This dataset contains video metadata and comments (along with their replies) from all over Australia, collected by performing search queries by geolocation. The collection of this dataset commenced on 2nd July 2021.

Each document is contains metadata for a YouTube video or is a comment thread for a video.

Data-centric activities this resource may assist with: #explore #collect


Organisations/Institutions: University of Melbourne

First published on: 02-07-2021
